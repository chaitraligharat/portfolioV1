Jul 2012 - Dec 2015

  • Developed server side component for company’s Information Rights Management product.
  • Led product release cycles and ran product development sprints in agile fashion.
  • Integrated product with existing single sign on based authentication systems for multiple clients using Ping Federate and SAML. This led to customized product solutions for clients.
  • Implemented user information management on web portal which allowed the end users to manage their accounts from a single portal.
  • Provided internationalization support for German, Russian and Spanish for the web portal. Consistent UI and UX across the web portal irrespective of the language of content displayed was achieved.
  • Simplified the login process for the product by a common login page across different user repositories like Simple AD, SSO repositories, etc. This was a huge leap from UX perspective for the product as the user login workflow shortened from confusing two-step process to single step.
  • Developed product integration with third party Document Management Systems. The policies defined in the third party DMS could automatically be applied at runtime to the documents downloaded.
  • Simplified user on-boarding for the product. Implemented workflows that ensured seamless onboarding and minimized drop rates when a new user interacted with the product. Extended the workflows to enrich product interaction for an existing user as well.
